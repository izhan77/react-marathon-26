# 🎬 MovieRealm - Movie Discovery Platform

> A modern, responsive movie discovery app built with React that lets you explore popular, top-rated, and upcoming movies with powerful filtering and sorting features.

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_App-4CAF50?style=for-the-badge&logo=vercel)](https://movierealm.vercel.app)

---

## 🎯 Overview

MovieRealm is a full-featured movie discovery platform that helps you browse, filter, and sort movies across different categories. Built with React and integrated with The Movie Database (TMDB) API, it provides a seamless, responsive experience for movie enthusiasts.

### Why I Built This

I built MovieRealm to master React concepts including React Router for navigation, API integration with fetch, dynamic filtering and sorting with Lodash, and environment variables for API security. This project represents my journey from learning React to building and deploying a production-ready application.

---

## ✨ Key Features

### 🎬 Movie Discovery
- **Popular Movies**: Browse currently trending films
- **Top Rated**: Discover critically acclaimed movies
- **Upcoming**: See what's releasing soon
- **Smooth Navigation**: React Router for seamless transitions

### 🔍 Filtering & Sorting
- **Rating Filter**: Filter movies by minimum rating (6+, 7+, 8+)
- **Sort Options**: Sort by release date or rating
- **Sort Order**: Choose ascending or descending order
- **Real-time Updates**: Instant filtering and sorting

### 🎨 User Experience
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Clean UI**: Modern, movie-themed interface
- **Active States**: Visual feedback for selected filters

### ⚡ Performance
- **Vite**: Lightning-fast development and builds
- **Optimized Dependencies**: Efficient bundling

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v14.0.0 or higher)
- **npm** or **yarn** package manager
- **Git** (for cloning)
- **TMDB API Key** (get it from [themoviedb.org](https://www.themoviedb.org))

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/izhan77/react-marathon-26.git
cd react-marathon-26/movierealm
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Set up environment variables**
```bash
# Create a .env file
echo "VITE_API_KEY=your_tmdb_api_key_here" > .env
```

4. **Start development server**
```bash
npm run dev
# or
yarn dev
```

5. **Open your browser**
```
http://localhost:5173
```

### Build for Production

```bash
npm run build
# or
yarn build
```

### Preview Production Build

```bash
npm run preview
# or
yarn preview
```

**Happy movie watching! 🎬**
