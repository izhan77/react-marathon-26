# 📋 TaskFlow - Task Management Dashboard

> A modern, interactive task management application built with React that allows you to organize your workflow with intuitive drag-and-drop functionality.

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_App-4CAF50?style=for-the-badge&logo=vercel)](https://task-trek-kappa.vercel.app)

---

## 🎯 Overview

TaskFlow is a full-featured task management dashboard that helps you organize your work across different stages of completion. Built with React and modern frontend practices, it provides a smooth, responsive experience for managing your daily tasks.

### Why I Built This

I built TaskFlow to master React fundamentals and implement complex user interactions like drag-and-drop. This project represents my journey from learning React concepts to building and deploying a fully functional application.

---

## ✨ Key Features

### 📝 Task Management
- **Create Tasks**: Add new tasks with titles and tags
- **Edit Tasks**: Modify existing tasks
- **Delete Tasks**: Remove tasks with one click
- **Categorize**: Tag tasks with HTML, CSS, JavaScript, or React

### 🔄 Drag & Drop
- **Intuitive Interface**: Drag tasks between "To Do", "Doing", and "Done" columns
- **Visual Feedback**: Drop zones highlight when dragging
- **Smooth Transitions**: Clean animations for better UX

### 💾 Data Persistence
- **Auto-Save**: Tasks automatically save to localStorage
- **Persistent State**: Your tasks persist across browser sessions
- **No Server Required**: Everything works offline

### 🎨 User Experience
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Clean UI**: Modern, minimalist interface
- **Visual Tags**: Color-coded tags for quick recognition
- **Real-time Updates**: Instant feedback on all actions
